
# Joke app

A monorepo consisting tiny full-stack app to fetch and modify joke from API.

## Stack

- Nest.JS, TypeScript
- Nuxt.JS 2, Vuetify 2, TypeScript
- Node Version (>= 16)

## Development

Run following commands to preview and develop

**Backend:**

```bash
cd api/
npm install
npm run start:dev
```

**Frontend:**

```bash
cd client/
npm install
npm run dev
```
